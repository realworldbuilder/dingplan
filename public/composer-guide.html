Construction Planner Composer Guide

Introduction

The Composer is a natural language interface that allows you to quickly create construction sequences from templates or build custom task schedules. This guide explains how to use the Composer effectively.

Using Templates

The system has modular templates organized by building type and construction phase:

  • Industrial Building Components: site_prep, foundation, steel, envelope, mep, equipment, interior, safety, utilities, commissioning
  • Specialized Structures: siteclearing, foundations, structure, facade, mep, systems, interiors, access, sitework, completion
  • General Components: foundation, steelstructure, mepzone, interiorzone, facadezone

"Create industrial foundation in Zone 1"

"Add specialized facade work to the project"

"Setup MEP for all zones"

Multi-Zone Implementation

Apply templates across multiple zones at once:

"Add industrial steel erection in all zones"

"Create MEP for each zone"

Use phrases like "in all zones" or "for each zone" to apply across zones.

Commands & Examples

To Do This Say This
List all templates "Show available templates"
View zones "List all zones"
Create single task "Create a framing task that takes 3 days"
Adjust timeline "Create a quick foundation template"
Delete task "Delete task [task ID]"

Tips & Tricks

  • Natural language: The system understands conversational requests
  • Keyword matching: Use industry terms to find relevant templates
  • Combining templates: Sequence templates to create complete project plans
  • Duration modifiers: Use words like "quick", "detailed", or "standard"
  • Specific dates: Include start dates like "beginning June 1st"

Template Sequence Example

Building a complete industrial building sequence:

  1. "Create industrial site prep in Zone 1 starting June 1st"
  2. "Add industrial foundation after site prep"
  3. "Create industrial steel erection following the foundation work"
  4. "Add industrial envelope after steel is complete"
  5. "Create industrial MEP rough-in after envelope"
  6. "Add interior work starting after MEP is 50% complete"
